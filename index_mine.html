House Project

This is the website for my computer science house project

By Amir Heidari

This website has three main sections:

  • Static House - This shows the original house that was developed in paint
  • Comparison - This is a shows my house in comparison to the house made following my instructions
  • Dynamic House - This is the final outcome after using processing to animate the house

Project:

  • This project had three parts
    • We had to make a house using paint and create instructions for it
    • Rereate the house in Processing and animate it
    • Create a website using html that displayed our house
  • During this project we learned about paint, Processing, and html
  • Processing and HTML
    • Processing is a compiled language and HTML is scripted
    • Both dont require spaces
    • HTML has open and closing braces
  • Processing and Paint
    • Paint is much easier to visualize because the picture is always infront of you
    • Paint is easier to use, no errors, never anything going wrong
    • Processing gives power to change and animate. Much more interactable
    • Processing Code vs. Paint instructions:
      • In Processing, or any programming language for that matter, there is no room for ambiguity whereas humans are able to interpret almost anything
      • Paint instructions are can be much simpilar and also much more human readable.
      • However, Processing is much more precise because a computer cannot make a mistake, the only thing that can be flawed is the code
